# Projektas
Filmų katalogo valdymo sistema

Projekto tikslas:
Sukurti C++ konsolinę programą, skirtą filmų katalogo valdymui. Programa leidžia saugoti, peržiūrėti, redaguoti ir šalinti filmų duomenis bei atlikti papildomą duomenų apdorojimą.

Temos aprašymas:
Programa skirta filmų katalogui valdyti. Visi filmai saugomi tekstiniame faile ir programos paleidimo metu nuskaitomi į atmintį. Vartotojas gali atlikti įvairius veiksmus per meniu principu veikiančią sąsają.

Programos funkcionalumas:
CRUD operacijos
Pridėti naują filmą.
Peržiūrėti visus filmus.
Redaguoti esamą filmą.
Pašalinti filmą.

Papildomos funkcijos:
Filmų paieška pagal žanrą.
Filmų rikiavimas pagal įvertinimą.
Vidutinio filmų įvertinimo skaičiavimas.

Naudojami failai:
main.cpp
Filmai.txt

Programos paleidimas:
Atidaryti projektą Visual Studio arba kitoje C++ kūrimo aplinkoje.
Įsitikinti, kad failas filmai.txt yra tame pačiame kataloge kaip programa.
Sukompiliuoti programą.
Paleisti programą.
Naudotis meniu pasirinkimais.
